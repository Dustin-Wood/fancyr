# fancyr

**Fancy Statistics for Correlational (r) Analyses** — an R package by Dustin Wood.

Provides functions for correlational and measurement analyses, including
reliability-adjusted correlation matrices, scale-centered correlations
(Cohen, 1969), clusters of validity-relevant items, dimensionality assessment,
data screening utilities (e.g., proportion of maximum possible standard
deviation), and data transformations (e.g., POMP scores and
scale-center-deviation scores).

## Installation

```r
# install.packages("devtools")
devtools::install_github("Dustin-Wood/fancyr")
```

## Vignettes

- [nFK Reference Cases](https://dustin-wood.github.io/fancyr/test_nFK.html) —
  worked examples and reference cases for `nFK()` (number of orthogonal factors
  in a set).
